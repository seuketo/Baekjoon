N명의 사람
ACM 수석 판사

=> S만큼의 몫을 나눠 가져야 함.
	ㄴ 총 인원 : N + 1(수석 판사)
	=> 1 100 입력
		ㄴ N = 1
		ㄴ 총 인원 = 2
		ㄴ 100 / 2 = 50

	=> 2 7 입력
		ㄴ N = 2
		ㄴ 총 인원 = 3
		ㄴ 7 / 3 = 2

	=> 10 10 입력
		ㄴ N = 10
		ㄴ 총 인원 = 11
		ㄴ 10 / 11 = 0

코드를 구성하는 데 조금 고민해야 할 부분
    ㄴ 이거 입력을 한번만 하면 되는 부분인건가?
    ㄴ 아니면 여러 번의 입력을 받을 수 있게 해야하는건가?
		=> EOF로 파일을 작성해야 한다.
	

Main()에 직접 파일을 작성하지 않는다.
	ㄴ 따로 클래스를 하나 더 작성하여 main에 불러오는 형식을 채택하는 편이 효율면에서 좋을듯 하다.
